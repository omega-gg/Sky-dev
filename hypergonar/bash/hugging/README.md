# [Bash](../README.md) Hugging Face generative tools

## Environment

- SKY_HUGGING_KEY=your_token

- SKY_URL_HUGGING_QWEN_ANGLE=your_endpoint_url

## Tools

- [start.sh](start.sh): Start a Hugging Face endpoint.
- [stop.sh](stop.sh): Stop a Hugging Face endpoint.
- [qwen-angle.sh](qwen-angle.sh): Change the angle of an image.
