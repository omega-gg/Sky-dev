# [Bash](../README.md) Luma generative tools

## Environment

- SKY_LUMA_KEY=your_api_key.

## Tools

- [start-end.sh](start-end.sh): Generate a video with start and end frame.
- [reframe.sh](reframe.sh): Reframe the video to a new ratio.
- [reframe-wide.sh](reframe-wide.sh): Reframe the video to 21:9.
